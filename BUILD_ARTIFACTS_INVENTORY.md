## Deployment Instructions

### Option 1: Vercel (Recommended)

```bash
# From project root
vercel --prod
```

**Result**: Live site deployed to Vercel CDN with automatic rebuilds on GitHub push

### Option 2: GitHub Pages

```bash
# Push build to gh-pages branch
git subtree push --prefix build origin gh-pages
```

**Result**: Live site at `https://<username>.github.io/hackathon-book`

### Option 3: Self-Hosted (Nginx)

```bash
# Copy build to web root
sudo cp -r build/* /var/www/html/

# Configure Nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/html;
    index index.html;

    # Fallback to index.html for client-side routing
    error_page 404 /404.html;
}
```

### Option 4: Self-Hosted (Apache)

```bash
# Copy build to document root
sudo cp -r build/* /var/www/html/

# Configure .htaccess
<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteBase /
    RewriteRule ^index\.html$ - [L]
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteRule . /index.html [L]
</IfModule>
```

### Option 5: Docker

```dockerfile
FROM nginx:alpine
COPY build/ /usr/share/nginx/html/
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

## Post-Deployment Support

### Common Issues & Solutions

**Issue**: Urdu text not rendering RTL
- **Solution**: Clear browser cache, verify RTL CSS loaded
- **Check**: DevTools â†’ Network â†’ Check for `styles.*.css`

**Issue**: Pages not found (404 errors)
- **Solution**: Configure server to fallback to `index.html` for SPA routing
- **Check**: Verify `404.html` in root; configure rewrite rules

**Issue**: Images/CSS not loading
- **Solution**: Check deployment preserves all files in `/assets/` and `/img/`
- **Check**: Verify file permissions are readable (644 for files, 755 for dirs)

**Issue**: Search not working
- **Solution**: Verify all HTML pages deployed; search index might be lazy-loaded
- **Check**: Try refreshing page; search loads on first interaction

**Issue**: Performance slow**
- **Solution**: Enable gzip compression on server
- **Check**: Verify CDN caching enabled; check Network tab for compression
